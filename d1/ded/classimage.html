§ ProcessBoundaryX()

void image::ProcessBoundaryX ( )

This method process verticle tiling boundaries.

This method will reprocess segments which touch the verticle tiling boundaries. Instead of processing tiling boundaries from left to right one by one, we use a strategy which can run parallely. Here is an example which explains the idea. For example we have 9 by 1 tiles ( we only consider verticle tiling boundaries here), there are 8 verticle boundaries in total, at first round, we process number 1,3,5,7 and in second round, we process number 2 and 6, third round number 4 and firth round number 8. At each round, the tiling boundaries can be reprocess parallely.
