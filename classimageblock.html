void imageblock::saveToDisk ( int  level)

Save the segmentation.

This method save the segmentation of the tile. Segments which are not touch any tiling boundaries will be save to the final segmentation. while others will be save to temperary file named "i indexX j indexY", (eg, a tile with indexX 3 and indexY 2, the temperary file is name as "i3j2")
